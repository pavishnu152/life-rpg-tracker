# Life RPG Habit Tracker

A gamified habit tracking web application that turns daily routines into a leveling system.

## 🚀 Live Demo

https://life-rpg-tracker.onrender.com

## ⚙️ Tech Stack

* Flask (Python Backend)
* React + TailwindCSS (Frontend)
* SQLite Database
* Gunicorn
* Render Cloud Deployment

## ✨ Features

* Habit checklist
* XP reward system
* Level progression
* Streak tracking
* Discipline score
* Habit history heatmap
* Mobile responsive UI

## 📱 How It Works

1. Open the app
2. Complete habits
3. Earn XP
4. Increase level and streak
5. Track progress daily
